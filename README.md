# Easy Carros App

Bem vindo ao projeto easy carros app.


## Preview

--future-image


## Project setup
```
yarn install
```

### Compilar e hot reload para desenvolvimento
```
yarn run serve
```

### Compilar e minificar para produção
```
yarn run build
```

### Executar testes
```
yarn run test
```

### Lintar arquivos
```
yarn run lint
```
